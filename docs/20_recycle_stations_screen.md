# 20. リサイクルステーション画面

> 関連: `REQUIREMENTS.md` §2.1 F4, §5.1（recycle-stations.json）
> ステータス: 未着手

## 目的

ペットボトル・ガラスびん等を持ち込めるリサイクルステーション（ア〜ク 8グループ）を一覧・地図表示する。**全グループ全市民が利用可能**な点を明示する。

## 完了条件

- 8グループそれぞれの開催日・場所が見られる
- 「次回開催日」が分かりやすく表示される
- 自分の地区に近いグループを推奨表示
- 地図で位置確認・経路案内ができる
