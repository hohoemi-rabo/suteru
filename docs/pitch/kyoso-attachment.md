# 〔提案書〕スマホカメラで「ごみの捨て方」が分かるアプリの県内横展開
### — 住民の分別支援と、市町村の問い合わせ負担軽減を、写真一枚で —

> 飯田市版の**動作する試作（MVP）が完成**。地区・収集日データの差し替えで**県内全域へ横展開**できます。
> 提案者: （団体名・代表者・連絡先 ※確定後に記入）／ 開発: ほほ笑みラボ

---

## 1. 解決したい課題（県内に共通）

「このごみ、どう捨てればいいの？」という分別の迷いは、**県内のどの市町村でも共通する住民の困りごと**です。

- 分別ルールは**市町村ごと・品目ごとに細かく**、住民（特に若年層・転入者）が迷いやすい。
- 結果として、**自治体への問い合わせ対応**という業務負担にもつながっている。
- 分別精度の低下は、**ごみ処理コストや 3R（リデュース・リユース・リサイクル）推進**の妨げにもなる。

## 2. 解決策（仕組み）

```
 ごみを撮影  →  AIが「品目」を特定  →  その市町村の公式ルールを表示
                                       （収集日・分別区分・出し方の注意点）
```

- **AIの役割は「品目名の特定」だけ**。分別ルールは**各自治体の公式データ**に基づいて表示します
  （＝AIに分別ルールを推測させない「**ハルシネーション・ゼロ**」設計。誤案内を出しません）。
- 文字検索・収集日カレンダー・施設情報も搭載。各画面に「ベータ版」「公式情報を参照」を明示。
- **プライバシー配慮**: 撮影画像はサーバーに保存せず判定後に破棄／**ログイン不要**。

## 3. 現状 — すでに「動くアプリ」があります（実現性）

- **飯田市版の MVP（試作）が完成し、実機で動作確認済み**（撮影→品目判定→ルール表示までの一連）。
- 飯田市公式の分別情報をもとに **品目 約590・10 地区**に対応。
- **地区・収集日データを差し替えるだけで他市町村へ展開できる構造**（横展開が容易）。

> **[スクショ挿入]** ① ホーム画面（カメラボタン）　② 撮影→判定結果　③ 収集日カレンダー
> （実機のスクリーンショットを2〜3枚）

## 4. 県内横展開のイメージ

```
 飯田市で実証（完成済み）  →  近隣市町村で実証  →  県内市町村へ順次展開
```

データ構造が共通のため、各市町村の公式分別データを反映すれば、**同じ仕組みを県内全域に広げられます**。

## 5. 県との共創でお願いしたいこと

1. **実証フィールドの提供** … 県内の複数市町村での実証にご協力ください。
2. **データ提供・標準化の調整** … 各市町村の分別区分・収集日データの提供や、様式の標準化に向けた調整。
3. **関係先へのおつなぎ** … 環境部門・市町村・関係団体（市町村会等）への橋渡し。

> 費用の補填は前提とせず、対話を通じて県と当方の双方にとって最適な進め方を一緒に探りたいと考えています。

## 6. 期待する効果（公益）

- 住民の**利便向上**（写真一枚で自己解決）と**分別精度の向上**
- **ごみ減量・3R 推進**への寄与
- 自治体の**問い合わせ業務の軽減**・住民広報の補助
- 将来像: 飯田 → 南信州 → 県内 への横展開。持続的な提供の形も対話の中で検討。
