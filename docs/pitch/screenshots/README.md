# スクリーンショット撮影指示（app-guide.md 用）

実機（preview APK）で以下を撮影し、**このフォルダに指定ファイル名で保存**してください。
保存すると `docs/pitch/app-guide.md` に自動で図が入ります（拡張子は `.jpg` / `.png` どちらでも可。現状は `.jpg` で投入済み）。
1画面で2枚見せたいときは `NN-name_2.jpg` の形で追加（例: `02-camera_2.jpg`＝判定中、`07-search_2.jpg`＝品目詳細、`11-settings_2.jpg`＝通知/データ）。

> 共通のコツ: 地区を設定済みにしてから撮る／通知バーの個人情報（時刻以外）は気にしなくてOK／
> 明るい画面で。例に使う品目は「分かりやすく代表的なもの」を選ぶと伝わりやすい。

| ファイル名 | 画面・状態 | 出し方 | 写ってほしいもの |
|---|---|---|---|
| `01-home.png` | ホーム | アプリ起動直後 | 現在の地区・「写真でしらべる」・文字検索バー・「収集カレンダー」ボタン・ベータ版表示 |
| `02-camera.png` | カメラ撮影画面 | ホーム→写真でしらべる | 中央の撮影枠＋右上の「**画像は保存されません**」表示（プライバシーの要） |
| `03-result.png` | 判定結果（ヒット例） | 何か分かりやすい物を撮る or 文字検索でヒット→開く | 品目名・分別カテゴリの色タグ・**次回収集日**・出し方の説明（例: ペットボトル等、収集日が出る品目が映えます） |
| `04-not-found.png` | 辞書外＋報告ボタン | 辞書に無さそうな物を撮る（変わった家電・雑貨など） | 「**辞書にありません**」＋「**この品目を報告する**」ボタン |
| `05-report-done.png` | 報告完了 | ④で「この品目を報告する」を押した後 | 「**ご報告ありがとうございます**」の表示 |
| `06-line.png` | LINE通知 | ⑤の直後、LINEアプリを開く | 届いた「🗑 未収録品目の報告 …」の通知（品目名・地区が見える状態） |
| `07-search.png` | 文字検索 | ホーム→文字検索バー、何か入力 | 入力欄＋ヒット一覧（例: 「かさ」「でんち」など複数ヒットする語） |
| `08-calendar.png` | 収集カレンダー（ポップアップ） | ホーム→「収集カレンダー」 | 月カレンダー・日曜始まり・色ドット・凡例まで（できれば凡例が見える位置で） |
| `09-schedule.png` | 収集日タブ | 下タブ「収集日」 | 「次の収集」ハイライト＋カテゴリ別 次回収集日 |
| `10-facilities.png` | 施設タブ | 下タブ「施設」 | 施設の住所・電話・リサイクルステーションへの導線 |
| `11-settings.png` | 設定 | 下タブ「設定」 | 地区・通知・プライバシーポリシー/利用規約の項目 |
