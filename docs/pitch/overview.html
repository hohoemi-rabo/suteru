これどう捨てる？（仮称） ベータ版

飯田市のごみ分別を、写真一枚で。

カメラで撮るだけで、そのごみの「飯田市での捨て方」が分かる Android アプリ。

市民向け・無料／開発: ほほ笑みラボ（飯田市）

解決したい課題

  • 「このごみ、どう捨てる？」は市民全般の悩み（特に若年層・転入者）。
  • 分別ルールは地区・品目ごとに細かく、市役所への問い合わせ負担にも。

仕組み

ごみを撮影 → AIが品目を特定 → 飯田市の公式ルールを表示
（収集日・分別カテゴリ・出し方の注意点）
  • AIは品目の特定だけ。ルールは公式データに基づき表示（＝誤案内を出さない設計）。
  • 文字検索・収集日カレンダー・施設情報も搭載。

現状 — 動くアプリが完成

  • 実機で動作する試作（MVP）が完成（撮影→判定→ルール表示まで）。
  • 飯田市公式データで 品目 約590・10 地区に対応（拡張可）。
  • プライバシー配慮：画像はサーバー保存なし・ログイン不要。
  • 各画面に「ベータ版」「公式情報を参照」を明示し信頼性に配慮。

飯田にとっての価値

  • 市民が写真一枚で自己解決 → 市役所の問い合わせ業務の軽減。
  • 分別の質の向上・ごみ減量に寄与。
  • 地元発。将来は 飯田 → 南信州 → 県内 へ横展開も視野。
