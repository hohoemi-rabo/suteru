# これどう捨てる？（仮称）
### 飯田市のごみ分別を、写真一枚で。

> **カメラで撮るだけで、そのごみの「飯田市での捨て方」が分かる Android アプリ。**
> 市民向け・無料。開発元: ほほ笑みラボ（飯田市）。

---

## 解決したい課題

- 「このごみ、どう捨てればいいの？」は **市民全般の日常的な悩み**。特に **若年層・転入者** に多い。
- 分別ルールは地区ごと・品目ごとに細かく、**市役所への問い合わせ負担**にもつながりやすい。

## 仕組み（シンプル）

```
ごみを撮影 → AI が「品目」を特定 → 飯田市の公式ルールを表示
                                    （収集日・分別カテゴリ・注意点）
```

- **AI の役割は「品目名の特定」だけ**。分別ルールは飯田市の公式データに基づいて表示します
  （＝AI に分別ルールを推測させない「ハルシネーション・ゼロ」設計）。
- 文字検索でも調べられ、収集日カレンダー・施設情報も搭載。

## 現状（動く MVP が完成）

- **実機で動作するアプリが完成**（撮影 → 判定 → ルール表示まで一連が動く）。
- 飯田市公式の分別データをもとに **品目約 590・10 地区**に対応（さらに拡張可能）。
- **プライバシー最優先**：撮影画像はサーバーに保存しない／ログにも残さない・**ログイン不要**。
- 各画面に「**ベータ版**」「**公式情報を参照**」を明記し、信頼性に配慮。

## 飯田にとっての価値

- 市民の「捨て方が分からない」を**写真一枚で自己解決** → 市役所の問い合わせ業務の軽減に寄与。
- 公式情報への導線を保ちつつ、**分別の質の向上・ごみ減量**にもつながる。
- **地元発の取り組み**。将来は 飯田 → 南信州 → 県内 への横展開も視野。
