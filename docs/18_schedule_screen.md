# 18. 収集日画面（F3）

> 関連: `REQUIREMENTS.md` §2.1 F3, §6.1
> ステータス: 未着手

## 目的

ユーザーの地区における、各カテゴリの **次回収集日** とこの先2〜4週の予定を一覧表示する。タブの2番目。

## 完了条件

- カテゴリ別の次回収集日が一目でわかる
- 「今日」「明日」「明後日」が特別表示される
- 地区未設定時はメッセージ + 設定画面への誘導
