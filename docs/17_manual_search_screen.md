# 17. 手動検索画面（F2）

> 関連: `REQUIREMENTS.md` §2.1 F2, §3.2（補助フロー）
> ステータス: 未着手

## 目的

カメラが使えない時の保険として、品目名や別名でテキスト検索できる画面を提供する。`items.json` の `name` と `aliases` を対象に部分一致検索する。

## 完了条件

- 検索バーに入力すると、リアルタイムで候補一覧が絞り込まれる
- 候補をタップすると結果画面（[[16_result_screen]]）に遷移
- ヒットしない時の挙動が親切（公式サイトリンク等）
- オフラインでも動く
