# 17. 手動検索画面（F2）

> 関連: `REQUIREMENTS.md` §2.1 F2, §3.2（補助フロー）
> ステータス: ✅ 完了（2026-05-17）

## 実装メモ

- `lib/text-search.ts` に純粋関数で `normalizeJa`（ひらがな→カタカナ吸収）と `searchItems`（スコア付き検索）を切り出した
- スコアリング: name 完全1000 > 前方800 > 部分600 / alias 完全500 > 前方400 > 部分300、同点は `name` の日本語ソート
- **Result 画面（16）が未実装**のため、候補タップは Bottom Sheet 風 Modal でインライン表示（カテゴリバッジ + instruction + warnings + 「詳しい画面は近日公開予定」注記）。16 完成後に Modal を `router.push('/result?...')` に差し替える

## 目的

カメラが使えない時の保険として、品目名や別名でテキスト検索できる画面を提供する。`items.json` の `name` と `aliases` を対象に部分一致検索する。

## 完了条件

- 検索バーに入力すると、リアルタイムで候補一覧が絞り込まれる
- 候補をタップすると結果画面（[[16_result_screen]]）に遷移
- ヒットしない時の挙動が親切（公式サイトリンク等）
- オフラインでも動く
