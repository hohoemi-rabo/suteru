# 19. 施設画面（F4）

> 関連: `REQUIREMENTS.md` §2.1 F4, §6.1
> ステータス: 未着手

## 目的

クリーンセンター・最終処分場・家電引取業者などの持ち込み施設を一覧表示し、地図と連絡先（電話発信）を提供する。

## 完了条件

- 施設一覧がリスト + 地図で見られる
- 電話番号タップで発信できる
- Google Maps への経路案内リンクが動く
