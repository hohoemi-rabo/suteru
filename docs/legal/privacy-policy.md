# プライバシーポリシー

**施行日**: 2026-05-17

ほほ笑みラボ（以下「当方」）は、Android アプリ「これどう捨てる？」（以下「本アプリ」）における利用者の情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。本アプリは飯田市のごみ分別を支援することを目的としており、利用者の個人を特定する情報の取得を最小化する設計です。

## 1. 適用範囲

本ポリシーは、本アプリの利用に伴って当方が取り扱う情報すべてに適用されます。本アプリから外部のサービス（飯田市公式サイト、地図アプリなど）へ遷移した場合、当該サービスのプライバシーポリシーが適用されます。

## 2. 取得する情報と利用目的

本アプリは、以下の情報を必要最小限の範囲で取得・利用します。

- **撮影画像**: カメラで撮影されたごみの画像は、品目を判定する目的で当方が運用する Cloudflare Workers（プロキシ）経由で Google Gemini API に送信され、判定後ただちに破棄します。当方のサーバーおよびログには一切保存しません。
- **位置情報**: 利用者が「現在地から地区を設定」等の操作を行ったときに限り、端末から OS の位置情報 API を通じて取得します。取得した位置情報は端末内で最寄り地区の判定にのみ使用し、当方のサーバーや外部サービスには一切送信しません。
- **端末識別子**: API のレート制限（短時間に大量のリクエストを送る悪用の防止）のため、端末内で生成した文字列を SHA-256 アルゴリズムでハッシュ化したものをリクエストヘッダーに付与します。個人を特定できる元の値は当方には送信されません。
- **設定値**: 利用者が選択した地区、通知の ON/OFF、通知時刻などは端末内の AsyncStorage に保存されます。当方のサーバーには送信されません。

## 3. 第三者提供

当方は、取得した情報を第三者に提供することはありません。ただし、画像判定のために Google Gemini API へ画像データを転送する場合があり、これは利用目的の範囲内での技術的な処理委託として行うものです。

## 4. 利用する外部サービス

本アプリは、機能提供のため以下の外部サービスを利用します。各サービスのプライバシーポリシーは各社のサイトをご確認ください。

- Google Gemini API（画像判定用、Google LLC 提供）
- Cloudflare Workers（API プロキシ・レート制限用、Cloudflare, Inc. 提供）
- Google Maps（住所タップ時の地図表示用、Google LLC 提供）

## 5. アクセス解析

本アプリは Google Analytics 等のアクセス解析ツールを導入しておりません。利用者の操作履歴を当方が収集・分析することはありません。

## 6. 利用者の権利

本アプリは利用者のアカウントや個人情報を保存していないため、削除手続きはありません。端末内の設定値は本アプリの設定画面にある「設定をリセット」、または端末の OS 設定からアプリのデータを削除することでいつでも消去できます。
